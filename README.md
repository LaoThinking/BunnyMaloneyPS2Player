# BunnyMaloneyPS2Player
A PS2 application made with PS2SDK to play all the episodes from the show, Bunny Maloney!

# The goal of this project was to see the capibility of the PS2 to utilise its hardware to make video player. In these moments, I kind of attached to Bunny Maloney a lot, so I decided to make it reality and make it real (probably more Sega fans than Playstation fans :") )
